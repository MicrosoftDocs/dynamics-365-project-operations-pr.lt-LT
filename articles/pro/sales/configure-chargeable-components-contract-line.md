<span data-ttu-id="a420e-105">Į projektais pagrįstą sutarties eilutę *įtraukti* komponentai ir *apmokestinami* komponentai.</span><span class="sxs-lookup"><span data-stu-id="a420e-105">A project-based contract line has *included* components and *chargeable* components.</span></span>

<span data-ttu-id="a420e-106">Įtrauktiems komponentams taikomos toliau nurodytos sąlygos.</span><span class="sxs-lookup"><span data-stu-id="a420e-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="a420e-107">Sąskaitų išrašymo būdo ir klientų išskaidymas</span><span class="sxs-lookup"><span data-stu-id="a420e-107">Billing method and customer splits</span></span>

<span data-ttu-id="a420e-110">Įtrauktų komponentų pogrupis gali būti pažymėtas kaip apmokestinamas naudojant lauką **Atsiskaitymo tipas**.</span><span class="sxs-lookup"><span data-stu-id="a420e-110">A subset of the included components can be marked as chargeable using the **Billing Type** field.</span></span> <span data-ttu-id="a420e-111">Laukas **Atsiskaitymo tipas** yra parinkčių rinkinys, leidžiantis toliau nurodytas reikšmes sutarties eilutės kontekste.</span><span class="sxs-lookup"><span data-stu-id="a420e-111">The **Billing Type** field is an option-set that allows the following values in the context of a contract line:</span></span>

  - <span data-ttu-id="a420e-113">Neapmokestinama</span><span class="sxs-lookup"><span data-stu-id="a420e-113">Non-chargeable</span></span>

<span data-ttu-id="a420e-115">Projekto sutarties eilutės apmokestinamumas apibrėžiamas užduotyse ir taikomas visoms į eilutę įtrauktoms operacijų klasėms.</span><span class="sxs-lookup"><span data-stu-id="a420e-115">Chargeability is defined on tasks for a project contract line and applies to all transaction classes included on the line.</span></span> <span data-ttu-id="a420e-116">Jei sutarties eilutės laukas **Įtraukti užduotis** yra tuščias arba nustatytas į parinktį **Visas projektas** , skirtukas **Apmokestinamos užduotys** nebus pateiktas.</span><span class="sxs-lookup"><span data-stu-id="a420e-116">If the **Include Tasks** field on a contract line is blank or set to \* *Entire project* , the **Chargeable tasks** tab will not be available.</span></span>

<span data-ttu-id="a420e-117">Projekto sutarties eilutės apmokestinamumas, apibrėžtas vaidmenyse, taikomas tik tipo **Laikas** operacijų klasei.</span><span class="sxs-lookup"><span data-stu-id="a420e-117">Chargeability defined on roles for a project contract line only applies to the **Time** transaction class.</span></span> <span data-ttu-id="a420e-118">Jei sutarties eilutės laukas **Įtraukti laiką** yra nustatytas į parinktį **Ne** , skirtukas **Apmokestinami vaidmenys** nebus pateiktas.</span><span class="sxs-lookup"><span data-stu-id="a420e-118">If the **Include Time** field on a contract line is set to **No** , the **Chargeable roles** tab will not be available.</span></span>

<span data-ttu-id="a420e-119">Projekto sutarties eilutės apmokestinamumas, apibrėžtas operacijų kategorijose, taikomas tik tipo **Išlaidos** operacijų klasei.</span><span class="sxs-lookup"><span data-stu-id="a420e-119">Chargeability defined on transaction categories for a project contract line only applies to the **Expense** transaction class.</span></span> <span data-ttu-id="a420e-120">Jei laukas **Įtraukti išlaidas** yra nustatytas į parinktį **Ne** , skirtukas **Apmokestinamos kategorijos** nebus pateiktas.</span><span class="sxs-lookup"><span data-stu-id="a420e-120">If the **Include Expenses** field is set to **No** , the **Chargeable Categories** tab will not be available.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="a420e-121">Projekto užduoties atnaujinimas į apmokestinamą arba neapmokestinamą</span><span class="sxs-lookup"><span data-stu-id="a420e-121">Update a project task as chargeable or non-chargeable</span></span>

<span data-ttu-id="a420e-122">Projekto užduotis gali būti apmokestinama arba neapmokestinama konkrečioje sutarties eilutėje, todėl galima nustatyti toliau nurodytą sąranką.</span><span class="sxs-lookup"><span data-stu-id="a420e-122">A project task can be chargeable or non-chargeable on a specific contract line which makes the following setup possible:</span></span>

<span data-ttu-id="a420e-123">Jei projekto sutarties eilutėje yra nurodyti **Laikas** ir tam tikra užduotis, **T1** su ja susieta kaip apmokestinama.</span><span class="sxs-lookup"><span data-stu-id="a420e-123">If a project-based contract line includes **Time** and a certain task, **T1** is associated to it as chargeable.</span></span> <span data-ttu-id="a420e-124">Jei antroje sutarties eilutėje yra **Išlaidos** , T1 užduotį galite susieti sutarties eilutėje kaip neapmokestinamą.</span><span class="sxs-lookup"><span data-stu-id="a420e-124">If there is a second contract line that includes **Expense** , you can associate the T1 task on the contract line as non-chargeable.</span></span> <span data-ttu-id="a420e-125">Todėl visas laikas, užfiksuotas užduotyje, yra apmokestinamas, o visos išlaidos – neapmokestinamos.</span><span class="sxs-lookup"><span data-stu-id="a420e-125">The result is that all of the time recorded on the task is chargeable and all expenses are non-chargeable.</span></span>
