<span data-ttu-id="4ea67-104">Įtraukdami išteklių vaidmenis, jūs tikriausiai jau susiejote su jais kai kuriuos įgūdžius.</span><span class="sxs-lookup"><span data-stu-id="4ea67-104">While you were adding resource roles, you probably already associated some skills to them.</span></span> <span data-ttu-id="4ea67-105">Kartkartėmis gali prireikti įtraukti papildomų įgūdžių į išteklių vaidmenį.</span><span class="sxs-lookup"><span data-stu-id="4ea67-105">From time to time, you might need to add additional skills to a resource role.</span></span>  
  
1.  <span data-ttu-id="4ea67-106">Pasirinkite **Project Service > Išteklių vaidmenys**.</span><span class="sxs-lookup"><span data-stu-id="4ea67-106">Go to **Project Service > Resource Roles.**</span></span>
